This script will enumerate and stop all instances of the "Microsoft SharePoint Foundation Incoming E-Mail" service and stop all instances of the service in the farm.  This script requires no parameters to be configured.

Once the script is executed, the "Microsoft SharePoint Foundation Incoming E-Mail" will be stopped on all servers in the farm.

More details regarding this script and its usage can be found in my blog post:
How to Stop All Instances of the Incoming E-Mail Service

Download the script for a commented version of the script.

PowerShell
Edit|Remove
powershell 
Get-SPServiceInstance | ? {$_.Typename -eq "Microsoft SharePoint Foundation Incoming E-Mail"} | Stop-SPServiceInstance -Confirm:$False
Get-SPServiceInstance | ? {$_.Typename -eq "Microsoft SharePoint Foundation Incoming E-Mail"} | Stop-SPServiceInstance -Confirm:$False
