VERIFY HEALTH OF DOMAIN CONTROLLERS

Comments

 

DC Time Sync

 

☐

On your workstation, open the following website to determine the correct local time:

http://www.time.gov/

Next, open an elevated command prompt and execute the following command and note the time reported:

net time \\MWTDC02

Perform the same command for the remaining DCs: MSODC01, MSODC02, MWTDC01, MWTDC02

Ensure the time difference between any two DCs and the internet source doesnt exceed 1 minute

 

 

DCDiag Report

 

☐

Open a command window with your advisor account

 

☐

Execute the command:

dcdiag /c /v /q /skip:OutBoundSecureChannels /s:MWTDC02

 

☐

Monitor output for warnings and errors

 

☐

Perform the tasks above for MSODC01, MSODC02, MWTDC01

 

DC Event Logs Review

 

☐

Open the Event Viewer on each domain controller above and inspect the event logs for any unusual events. Be sure to review DHCP and DNS event logs.
