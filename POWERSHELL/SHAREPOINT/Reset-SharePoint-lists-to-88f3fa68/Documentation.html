Every once in a while, you have the need to reset SharePoint lists with custom permissions, to inherit permissions from the parent site instead.  This is especially useful during migration efforts from DocuShare or eRoom.  Though this can be done via the UI, it can be very combersome.  That's where this script comes into play.  It automates the process for you.

Arguments:

  1. URL - The site collection URL to target.
  2. List Title - The name of the list you wish to target.
  3. -all - The script will stop processing once it finds a list that matches.  If you are trying to process all lists by the same name e.g. "Shared Documents" you can add the -all switch to have it process the enditre site collection and reset all matching lists.
